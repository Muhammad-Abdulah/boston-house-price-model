# 🏠 Boston House Price Prediction API

An end-to-end Machine Learning project that predicts Boston house prices using a Random Forest Regressor and serves predictions through a FastAPI backend.

---

## 🚀 Features

- Machine Learning model trained using Scikit-learn
- Random Forest Regressor for regression task
- Data preprocessing and feature selection
- REST API built with FastAPI
- JSON-based prediction endpoint
- CORS enabled for frontend integration

---

## 🧠 Model Details

- Algorithm: Random Forest Regressor
- Features:
  - rm (average number of rooms)
  - lstat (lower status population %)
  - ptratio (pupil-teacher ratio)
  - nox (air pollution level)
  - dis (distance to employment centers)
  - tax (property tax rate)
  - age (older buildings percentage)

- Target: Median house price (medv)
